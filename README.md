# HandI 2022

### To get started

`git clone https://github.com/HAndIBrothers/Handi2022.git`

`cd handi2022`

`npm install`

`npm run dev`

http://localhost:8080/

### Development mode

`npm run dev`

### Build

` npm run build`

### Check file size(under 13kb)

Use after build
`npm run check-file-size`
